1. **Clone the repo**

npm install

3. **Set environment variables**

Create a .env file at the root of your project:

```bash
JWT_SECRET=<your-jwt-secret>
MONGODB_URI=<your-mongodb-uri>
PORT=<your-port>
OPENROUTER_API_KEY=<your-openrouter-api-key>
```

4. **Run the development server**

```bash
npm run dev
```
